Getting Started

This project enables you to rapidly bootstrap a Skyrim SKSE plugin using CommonLibSSE, supporting both the PO3 and NG forks.

What to Expect

After downloading the repository, run the included bootstrap script CMake_Build.bat. This script will prompt you to configure your new plugin project by entering the following:

  • CommonLibSSE variant: Choose between the powerof3 (PO3) or alandtse (NG) fork
  • Project Name: Your pluginâ€™s name
  • Author: Your name or handle
  • Description: A short description of your plugin
  • Version: Example: 1.0
  • Year: For copyright, e.g., (C) 2025
  • License: Example: MIT
  • Template Type: Select one of the following templates to use as a starting point:
    • Bare: Minimal setup with essentials to get your plugin running
    • Basic: Includes a console hook and event registration for enhanced interaction
    • Menu: Builds on Basic by adding an ImGui-powered menu interface for easier debugging and controls

Optional Dependencies

Some useful libraries are included but disabled by default. You can activate them by uncommenting their entries in submodule_config.cmake:

  • ImGui: Provides UI support included only with the Menu template. This template contains the necessary boilerplate code to get ImGui menus running smoothly.
  • JSON (nlohmann-json): JSON parsing and serialization
  • SimpleIni: Lightweight INI file parser
  • TOML: TOML file format support

Activating these lets you add powerful features without manual setup.
